# Examples

Build the project first:

```bash
npm run build
```

## Structure

```
01-getting-started/  # Start here if you're new
02-features/        # Individual features (math, tables, code, etc.)
03-advanced/        # Options, error handling, workflows
04-use-cases/       # Real documents (academic paper, blog post, docs)
```

## Getting Started

```bash
# Simplest example
node examples/01-getting-started/hello-world.js

# Convert a markdown file
node examples/01-getting-started/basic-formatting.js
```

## Feature Examples

Test any feature with:

```bash
node examples/02-features/convert-feature.js <filename.md>
```

Available: `math.md`, `tables.md`, `code-blocks.md`, `footnotes.md`, `frontmatter.md`, `links-images.md`

The features subdirectory has its own README with details.
