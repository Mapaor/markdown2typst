# Feature Examples

These examples demonstrate individual features of markdown2typst.

## Running the Examples

Each example can be tested using the generic converter:

```bash
node examples/02-features/convert-feature.js <filename.md>
```

For example:
```bash
node examples/02-features/convert-feature.js math.md
```

## Featured in theses examples
 - Mathematics — inline & block LaTeX
 - Tables — headers and column alignment
 - Code blocks — syntax-highlighted examples
 - Footnotes — references and long notes
 - Frontmatter — YAML metadata (title, authors, date)
 - Links & Images — inline/reference links and images


## Implementation Notes

All features are based on GitHub Flavored Markdown (GFM) specifications and are fully compatible with standard Markdown syntax and standard YAML Markdown Front-matter.
