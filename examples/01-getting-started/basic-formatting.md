# Basic Formatting Example

This document demonstrates the most common Markdown formatting features.

## Text Styles

You can make text **bold**, *italic*, or ***both***.

You can also use ~~strikethrough~~ text.

## Lists

Unordered list:
- First item
- Second item
- Third item

Ordered list:
1. Step one
2. Step two
3. Step three

## Links and Code

Visit the [Typst website](https://typst.app/) for more information.

Inline code: `let x = 42`

## Blockquote

> This is a blockquote.
> It can span multiple lines.
